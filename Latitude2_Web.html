Latitude vs X

The purpose of the project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled the data from the OpenWeatherMap API to assemble a dataset on 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of weather vs latitude. Factors we looked at included: temparature, cloudiness, wind speed and humidity. This site provided the source data and visualizations created as part of the analysis as well as explanations and descriptions of any trends and correlations witnessed.
